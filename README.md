# MockU 🚀

AI-based mock test platform for competitive exams like JEE, SSC, GATE, etc.

## ✨ Features
- Real-time exam simulation
- Section-wise performance analysis
- Time tracking per question
- Dashboard with insights

## 🛠 Tech Stack
- React.js
- Node.js
- Express
- MongoDB
